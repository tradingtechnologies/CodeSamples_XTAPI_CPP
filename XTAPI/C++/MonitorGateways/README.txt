MonitorGateways (C++)
===============================================================================


Overview
-------------------------------------------------------------------------------

This example demonstrates using the XTAPI to monitor the status of TT Gateways.


Instructions
-------------------------------------------------------------------------------

1. Upon launching, all of the TT Gateway names will be populated in the 
   "Available Gateways"combo box.
2. Select the TT Gateway to open.
3. Check the Price, Order and Fill boxes for the servers to open.
4. Click the Connect button. 
5. The Gateway Status groupbox will update with the state change.
6. Verify the connection in the Application Tree in Guardian.
 

XTAPI Objects
-------------------------------------------------------------------------------

TTGate
