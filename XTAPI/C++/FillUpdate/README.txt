FillUpdate (C++)
===============================================================================


Overview
-------------------------------------------------------------------------------

This example demonstrates using the XTAPI to retrieve fills for the
authenticated X_TRADER user.


Instructions
-------------------------------------------------------------------------------

1. No user interaction required.  The fill download will begin when the 
   application is loaded.


XTAPI Objects
-------------------------------------------------------------------------------

TTOrderSet
TTFillObj
